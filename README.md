## 📱 Clone do Instagram

Projeto desenvolvido com o objetivo de recriar a interface do Instagram, focando em design moderno, responsividade e boas práticas de desenvolvimento front-end.

---

## 🚀 Demonstração

💻 Projeto disponível

📌 (https://clone-de-instagram.vercel.app/)

---

## 🧠 Objetivo do Projeto

Este projeto foi desenvolvido com foco em:

Reproduzir uma interface real (UI Clone)
Praticar estruturação com HTML5
Aplicar estilizações modernas com CSS3
Trabalhar com interatividade usando JavaScript
Evoluir na organização de projetos front-end
Utilizar Git e GitHub no fluxo profissional

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

---

## 📱 Funcionalidades

Interface inspirada no Instagram
Layout moderno e fiel ao design original
Alternância de imagens na tela (efeito dinâmico)
Estrutura organizada em pastas (assets, scripts, estilos)
Responsividade básica para diferentes telas

---

## 🎨 Destaques do Projeto

Uso de imagens realistas simulando a interface do app
Estilização detalhada com foco em UI/UX
Código limpo e bem organizado
Projeto ideal para portfólio front-end

## 📈 Aprendizados

Manipulação do DOM com JavaScript
